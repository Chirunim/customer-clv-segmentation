# customer-clv-segmentation
This project builds an end to end customer segmentation and lifetime value pipeline for an energy retail business. It generates and analyses 100,000 domestic customer records,  calculates RFM scores (Recency, Frequency, Monetary value) and Customer Lifetime Value for each account, segments customers into four behavioural groups, and identifies which high value customers are at risk of churning. The output is a Power BI dashboard and plain English segment profile report that gives commercial and marketing teams a data driven basis for prioritising retention spend rather than treating all customers the same. All data is synthetically generated to mirror realistic energy retail customer behaviour.
